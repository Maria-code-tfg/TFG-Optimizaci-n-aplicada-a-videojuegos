# TFG-Optimizacion-aplicada-a-videojuegos
Repositorio de implementaciones de los algoritmos descritos en el TFG Optimización aplicada a videojuegos. Aquí iré subiendo los avances en dicho TFG.
Hasta ahora están subidos los algoritmo de Dijkstra, A* y D*Lite, junto con estructuras de datos que utilizan,como colas indexadas y cuadrículas, además de diferentes herramientas para comprobar su funcionalidad y generar gráficas.
